
# OSMsoft员工薪资管理系统  

## 什么是OSMsoft员工薪资管理系统  

OSMsoft是依据《企业级软件开发实验课》的课程要求实现的企业员工薪资管理系统。  

## OSMsoft有什么功能  

### OSMsoft包含的两种用户类型  

OSMsoft薪资管理系统提供两种用户类型  
1.管理员$(Admin)$  
2.员工$(Employee)$  

### 不同的用户类型可以实现的功能  

对于**管理员**而言，能实现的功能有  
1.对员工部门实现增删改查  
2.对员工实现增删改查  
3.修改员工的薪资  
4.按要求查询部门或员工  

对于**员工**而言，能实现的功能有  
1.修改个人信息  
2.按需求查看个人薪资
